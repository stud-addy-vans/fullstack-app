# Full Stack Auth App

## Features
- User Signup & Login
- JWT Authentication
- MongoDB Database
- Protected Dashboard
- Dark Mode UI

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js + Express
- Database: MongoDB Atlas
