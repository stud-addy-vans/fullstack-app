## 📌 Features

* 🔐 User Signup & Login (JWT Authentication)
* 🔑 Password hashing using bcrypt
* 🗄️ MongoDB database integration
* 📊 Dashboard with user session
* 👤 Profile page with real user data
* 🌙 Dark mode toggle
* ⚡ Responsive admin-style UI
* ✅ Form validation (no empty submissions)

---

## 🛠️ Tech Stack

### Frontend

* React (Vite)
* Axios
* React Router

### Backend

* Node.js
* Express.js
* JWT Authentication

### Database

* MongoDB Atlas

### Deployment

* Frontend: Vercel
* Backend: Render

---

## ⚙️ How It Works

1. User signs up → data stored in MongoDB
2. Password is hashed using bcrypt
3. On login → JWT token is generated
4. Token + user data stored in localStorage
5. Dashboard & Profile use stored user session
